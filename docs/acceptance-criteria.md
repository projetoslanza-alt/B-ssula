# Critérios de Aceite — Primeira Entrega

## Autenticação e tenant
- [ ] Usuário autentica
- [ ] Usuário pertence a organização
- [ ] Tenant A não acessa tenant B

## Conteúdo
- [ ] Admin cria categoria
- [ ] Admin cria e publica curso
- [ ] Curso visível para público autorizado
- [ ] Curso oculto para não autorizado

## Aprendizagem
- [ ] Usuário inicia curso (sem "matrícula" na UI)
- [ ] Progresso persistido
- [ ] Vídeo retoma posição
- [ ] Continuar de onde parou

## Gestão
- [ ] Treinamento obrigatório com prazo
- [ ] Status atrasado
- [ ] Gestor atribui curso

## Qualidade
- [ ] Layout responsivo
- [ ] typecheck, lint, testes, build passam
- [ ] Deploy documentado
